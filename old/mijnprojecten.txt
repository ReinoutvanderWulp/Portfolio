            <h1>DevOps Minecraft App project</h1>
            <figure class="figure">
                <img src="images/app.png" class="img" alt="app" title="app">
                <figcaption class="figure-caption">Screenshot van het eindresultaat van de minecraft-app</figcaption>
            </figure>
            <figure class="figure">
                <img src="images/output.png" class="img" alt="output" title="output">
                <figcaption class="figure-caption">Screenshot van de output van de database in de app</figcaption>
            </figure>
            <p>Voor het vak DevOps and security heb ik een case study moeten maken. Er waren verschillende opties, ik
                koos voor een applicatie
                maken met .NET in C# met windows form, en een database connectie, in mijn geval Sqlite. Bovenstaande
                afbeeldingen geven een idee
                hoe de app werkt. Het idee van de app is dat de gebruiker data uit een database kan opvragen. Deze data
                is info over de
                verschillende blokken/monsters/planten van minecraft. De gebruiker kan zelf ook data toevoegen. Alle
                data zit opgeslagen in een
                database, een SqliteDB. Het leuke aan dit project is dat ik met windows form heb gewerkt, wat ik nu in
                het vak .NET Development
                ook nodig heb. Verder heb ik geleerd te werken met C# in .NET met een DB. Dit project heeft me niet
                alleen veel
                dingen bijgebracht voor het vak DevOps, maar ook een goede intro gegeven voor .NET Development</p>
